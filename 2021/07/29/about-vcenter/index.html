二、分布式交换机配置 (DSwitch)

按下面步骤操作：

  1. 在 DSwitch 上点右键，选择 Add and Manage Hosts...
  2. 选择 Add hosts
  3. 点击 New hosts...
  4. 选择要加入的主机，打上勾
  5. 设置上行端口，选择一块网卡，点击 Assgin uplink
  6. 勾上 Apply this uplink assignment to the rest of the hosts 可以把其余主机一样设置
  7. 点 NEXT
  8. 继续 NEXT
  9. 不要 Migrate 后面手工搞比较安全
  10. 确认后点 FINISH

    三、配置分布式端口组属性

  11. 在分布式端口组上点右键，选择 Edit Settings...
  12. 选择 VLAN，在 VLAN type 中选择 VLAN，在 VLAN ID 中填入vlan号
